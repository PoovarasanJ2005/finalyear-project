BloodGroup AI

Advanced Fingerprint-Based Detection System

About This Project

Welcome to the BloodGroup AI platform. This is a complete Web + Machine Learning system that identifies blood groups based on fingerprint topology. The backend leverages a Convolutional Neural Network (CNN) trained on thousands of samples from biometric datasets to establish correlations between specific ridge patterns and biological markers.

How It Works

  • 1. Secure Entry: Users authenticate via JWT (JSON Web Tokens) ensuring all biometric operations are confidential.
  • 2. Image Upload/Scan: Medical personnel can upload legacy images or run a simulated real-time optical scan.
  • 3. ML Inference: The backend pushes the image data to our PyBrain (TensorFlow/Keras model), which extracts spatial hierarchies to deduce the blood group.
  • 4. Secure Logging: Scan metadata, timestamp, user, and confidence are logged to a protected embedded database.

Upload Requirements

  • Extensions: .JPG, .PNG, .BMP
  • Size: Recommended < 5 MB
  • Quality: Ensure clear, unblurred, top-down fingerprint scans with high contrast for optimal AI accuracy (>90%).
